# Welcome to FitPlanner

**FitPlanner** is a visually appealing and user-friendly fitness dashboard designed to showcase fitness plans, progress, and insights. This project aligns with modern UI trends, ensuring a professional, engaging, and responsive design.

### **Technologies and Libraries**
1. **React**: For building the dynamic and interactive user interface.
2. **Tailwind CSS**: For utility-first, responsive styling.
3. **Axios**: To fetch data dynamically from an API.
4. **React Router**: For seamless navigation across pages.
5. **React Awesome**: To create a motivational carousel with animations.
6. **Recharts**: For visualizing progress with interactive charts.
7. **React Icons**: For professional-looking icons.

### **Design Tools**
- Color Palette: Gradient from Pink (#FF6F91) to Blue (#4A90E2).
- Font Family: `Poppins` for headings and body, `Roboto Slab` for motivational quotes.
- Background and Shadow Effects: Subtle grays and glowing hover animations.
